05 Teach : Activity - Naive Bayes

Objective

Understand the basics of the naive Bayes algorithm.

Please note that this is not a programming assignment. Instead, you will walk through the basics of the algorithm using a spreadsheet.

In case it is helpful, here is a link to the spreadsheet from the class example: https://docs.google.com/spreadsheets/d/13KRe9-Yle7YdJR9ozw_RRP09aKfc2GeRd38s8SULDnc/edit?usp=sharing
