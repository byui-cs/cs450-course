Machine Learning & Data Mining | CSE 450

CSE 450 Team Project Requirements

Overview

The semester long team project will provide a chance for you to demonstrate mastery of the principles of machine learning and data mining.

In teams of three (two if necessary, rather than four), you will select your own data mining project and work on it throughout the semester to discover something interesting.

Evaluation Criteria

The following are the line items of the final project rubric. You should choose a project that is meaningful/challenging enough for you to demonstrate mastery in each of these areas:

  • Demonstrates understanding of the business needs / context of the data

  • Demonstrates ability to handle non-trivial dataset

    • (e.g., data set size, complexity of features, missing data, redundant attributes, obtaining the data, etc.)

  • Demonstrates proper algorithm selection and application

  • Discovered something interesting / of value

    • (e.g., about the data, about the algorithms and their limitations in this context, something actionable for a stakeholder, etc.)

  • Demonstrates understanding of limitations of the solution and potential ethical issues

  • Conversant in appropriate terminology and technologies

Mastery will be observed throughout the semester and specifically demonstrated via a final project report.
