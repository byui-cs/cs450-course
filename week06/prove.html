Neural Network Part 1

Objective

Understand Neural Networks, in particular, the back-propagation algorithm for multi-layer perceptron networks.

Please note that you have 3 weeks to implement and/or experiment with this algorithm. This is because it can be a challenge! This is likely the most challenging individual programming assignment we will do this semester. Do not put this off until later!

Choose Your Own Adventure

As with the Decision Tree assignment, you may choose either the Implementation Option or the Experimentation option for the Neural Network assignment:

  • Implementation Option - Build a multi-layer perceptron neural network that uses backpropagation to learn. Use this network to make predictions on a dataset of your choice and compare to an existing, off-the-shelf library implementation.

  • Experimentation Option - Use an existing, off-the-shelf library implementation of a multi-layer neural network to make predictions on several different datasets, and experiment with several different hyper-parameter combinations, comparing and contrasting their results.

Implementation Option

You will be adding to your experiment shell from the previous assignment, by implementing a multi-layer perceptron neural network.

You will finish the assignment in future weeks, however, to help you make progress, you need to have at least the following completed this week:

  1. Create some type of mechanism to represent your nodes (i.e. neurons) and their associated weights.

  2. Provide a way to create a single layer of nodes of any number (this should be easily specified via a parameter).

  3. Account for a bias input.

  4. Be able to take input from a dataset instance (with an arbitrary number of attributes) and have each node produce an output (i.e., 0 or 1) according to its weights.

  5. Be able to load and process at least one dataset of your choice.

  6. You should appropriately normalize the data set.

Notice that at this point, we are NOT implementing any sort of training or learning in our algorithm. The point of this part of the assignment is to make sure you have the basics of a network of nodes in place that can handle a real instance from a dataset.

Next week, we will extend this classifier to handle multiple layers of nodes, and implement the back-propagation algorithm for updating weights. Then, it will be much more interesting ;-).

Experimentation Option

Make progress toward your final experiments by completing the following this week:

  1. Load at least one dataset of your choice. You should choose something more interesting than loading the built-in Iris dataset.

  2. Appropriately prepare your dataset for a neural network, which may include handling missing data, normalizing numeric values, etc.

    Also, make sure to split your dataset into a training and test set.

  3. Identify a neural network library and learn about its parameters and how to use it.

  4. Train a neural network with your dataset.

  5. Use your network to make predictions about a test set and note the success of the algorithm on your dataset.

  6. Experiment with at least 3 different sets of hyper-parameters for the algorithm (e.g., different number of layers and nodes per layer, different learning rates, different activation functions, etc.).

General Guidelines

Because we are not finishing the assignment at this point, this assignment does not specifically require (or reward) going above and beyond. That will be addressed in a future assignment.

You are, of course, encouraged to go beyond these expectations because that will help you finish the complete assignment.
